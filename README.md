# 电气专业选修课

![考查课](https://img.shields.io/badge/%E8%80%83%E6%9F%A5%E8%AF%BE-green)

电气专业的选修课开设于第三学年春季学期和第四学年秋季学期。自 22 级起，电气专业的选修课不再区分*电力系统及其自动化*和*电力电子与电机驱动*两个方向，而是从全部的22门课程中选择任意门修读满足学分要求即可。22 级要求 9 学分，23 级要求 8 学分，24 级要求 10 学分。
